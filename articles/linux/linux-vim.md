# Linux Vim 操作

## 模式

Vim 有三种模式：
- **命令模式**：默认模式，执行命令
- **编辑模式**：输入文本
- **末行模式**：执行保存退出等操作

## 光标移动

| 快捷键 | 说明 |
|--------|------|
| h/← | 左移 |
| j/↓ | 下移 |
| k/↑ | 上移 |
| l/→ | 右移 |
| 0 | 移动到行首 |
| $ | 移动到行尾 |
| gg | 移动到第一行 |
| G | 移动到最后一行 |
| nG | 移动到第 n 行 |
| Ctrl+f | 向下翻页 |
| Ctrl+b | 向上翻页 |

## 可视模式

| 快捷键 | 说明 |
|--------|------|
| v | 字符可视模式 |
| V | 行可视模式 |
| Ctrl+v | 块可视模式 |

块可视模式下：
- `I` 插入
- `d` 删除
- 多行注释常用

## 编辑操作

| 快捷键 | 说明 |
|--------|------|
| dd | 剪切当前行 |
| ndd | 剪切 n 行 |
| yy | 复制当前行 |
| nyy | 复制 n 行 |
| p | 粘贴到下一行 |
| P | 粘贴到上一行 |
| u | 撤销 |
| Ctrl+r | 重做 |

## 插入模式

| 快捷键 | 说明 |
|--------|------|
| i | 光标前插入 |
| I | 行首插入 |
| a | 光标后插入 |
| A | 行尾插入 |
| o | 下一行插入 |
| O | 上一行插入 |

## 查找替换

```bash
/keyword          # 查找，n 下一处，N 上一处
:s/old/new       # 替换当前行第一个
:s/old/new/g     # 替换当前行全部
:%s/old/new/g    # 全局替换
```

## 保存退出

| 命令 | 说明 |
|------|------|
| :w | 保存 |
| :q | 退出 |
| :wq | 保存并退出 |
| :q! | 强制退出 |
| :wq! | 强制保存并退出 |

## 其他

```bash
:set nu          # 显示行号
:set nonu        # 取消行号
:! command       # 执行 shell 命令
```

## 常见问题

### 交换两行

1. 将光标移动到要交换的第一行
2. dd 剪切
3. p 粘贴到下一行

### 删除所有内容

```bash
ggVGd            # 全选并删除
```
