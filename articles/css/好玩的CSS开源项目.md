# <font style="color:rgb(255, 104, 39);">01</font>
**<font style="color:black;">Font-Awesome</font>**

<font style="color:rgb(61, 61, 61);">这个 70K  Star 的开源项目是一个 CSS、SVG、字体的工具箱。在这里你可以找到你任何想要的可缩放矢量图标，它可以被定制大小、颜色、阴影以及任何可以用 CSS 的样式，已经被数百万设计师、开发者使用。</font>

<font style="color:rgb(43, 43, 43);background-color:rgb(248, 248, 248);">开源地址：https://github.com/FortAwesome/Font-Awesome</font>

# <font style="color:rgb(255, 104, 39);">02</font>
**<font style="color:black;">Animate.css</font>**

<font style="color:rgb(61, 61, 61);">相信每一位前端工程师在写动画的时候，都会想到去 GitHub 上去看看有没有现成的轮子可用。开源项目 Animate.css 算的上是 GitHub 山最受欢迎的 CSS 动画库。</font>

<font style="color:rgb(61, 61, 61);">Animate.css 支持所有主流的浏览器和操作系统，如今已经 75K 的 Star，大多数动画效果你只需要简单的两行代码就能完成，比如弹跳、淡入淡出、抖动、闪现等等。</font>

<font style="color:rgb(43, 43, 43);background-color:rgb(248, 248, 248);">开源地址:https://github.com/animate-css/animate.css</font>

# <font style="color:rgb(255, 104, 39);">03</font>
**<font style="color:black;">normalize.css</font>**

<font style="color:rgb(61, 61, 61);">如果想让不同的浏览器在渲染网页元素的时候形式更统一，Normalize.css 是一个不错的选择，这个开源项目在 GitHub 上斩获 48.5k 的  Star。</font>

<font style="color:rgb(61, 61, 61);">相比于传统的 CSS reset，Normalize.css 是一种现代的、为 HTML5 准备的优质替代方案。</font>

<font style="color:rgb(43, 43, 43);background-color:rgb(248, 248, 248);">开源地址：https://github.com/necolas/normalize.css</font>
