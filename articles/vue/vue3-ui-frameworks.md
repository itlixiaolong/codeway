# Vue 3.0 框架集合

Vue3 已经发布一段时间了，版本趋于稳定并且可以用于生产。本文推荐几个比较流行的 Vue 3 UI 框架。

## Ant Design Vue

[官方网站](https://2x.antdv.com/components/overview/)

Ant Design Vue 是一个非常成熟的框架，组件适应各种图标样式、字体和黑色主题。不断改进其 60 多个组件，基本覆盖项目大部分需求。

**特点**：
- 支持 SSR 和组合 API
- 包含复杂组件如数据表、确认框、模态框
- GitHub 15k+ 颗星

## BalmUI

[官方网站](https://next-material.balmjs.com/)

BalmUI 基于谷歌的 Material Design，提供 Vue 插件和指令，以及高度可定制的组件。

**特点**：
- 支持复杂的内置指令
- 包含 debouncing 和 UI 波纹等效果

## Wave UI

[官方网站](https://antoniandre.github.io/wave-ui/)

Wave UI 是首批 Vue 3 UI 框架之一，提供 40+ 漂亮且响应迅速的组件。

**特点**：
- 组件非常漂亮，动画效果出色
- 企业级响应式 Vue 3 应用不错的选择

## Vuestic

[官方网站](https://vuestic.dev/)

Vuestic 是 Vue 最漂亮的开源管理面板之一，提供 50+ 组件。

**特点**：
- 优秀键盘导航支持
- 响应式设计，几乎适用于所有屏幕分辨率
- 无缝翻译支持

## Element Plus

[官方网站](https://element-plus.org/)

Element Plus 为 Vue 3 带来了大量组件，提供成熟的日期选择器、树、时间线等。

**特点**：
- GitHub 11.1k+ 颗星
- 高定制性，通过 SCSS 变量
- 成熟的国际化系统

## Ionic

[官方网站](https://ionicframework.com/docs/vue/overview)

Ionic 是最早提供 Vue 3 支持的 UI 框架之一，更倾向于移动 UI。

**特点**：
- GitHub 45k+ 颗星
- 大公司使用（空客、艺电）
- 成熟的社区支持

## Naive UI

[官方网站](https://www.naiveui.com/)

Naive UI 由 Vue 创建者转发推荐，三个月内 GitHub 5.1k 颗星。

**特点**：
- 70+ 制作精良的组件
- 性能优异，可定制性极强
- 支持 TypeScript

## Quasar

[官方网站](https://next.quasar.dev/)

Quasar 是一个完整的、以性能为中心的框架，可帮助构建 Vue 用户界面（SPA、PWA、SSR、移动和桌面）。

**特点**：
- 包含 Cordova、Capacitor 和 Electron
- 基于 Material Design
- 快速发布 Vue 3 支持

## PrimeVUE

[官方网站](https://primefaces.org/primevue/)

PrimeVUE 拥有 80+ 组件，是组件范围最广的框架之一。

**特点**：
- 包含仪表、滑块、树、图形等
- 集成 Vuelidate 表单验证
- 提供可视化编辑器

---

随着生态系统的迁移，从 Vue 2 迁移到 Vue 3 将变得更加容易。
