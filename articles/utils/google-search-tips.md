# Google 搜索技巧

## 1. 精确匹配

使用引号 `" "` 进行精确匹配：

```
"前端工程师"
```

## 2. 排除关键词

使用减号 `-` 排除：

```
前端 -后端
```

## 3. OR 搜索

使用 `OR` 或大写 OR：

```
Vue OR React
```

## 4. 站内搜索

使用 `site:` 限定站点：

```
site:github.com vue
site:juejin.cn 前端
```

## 5. 文件类型

使用 `filetype:` 搜索特定文件：

```
filetype:pdf 前端面试题
filetype:doc 技术文档
```

## 6. 链接搜索

使用 `link:` 查找链接到某页面的页面：

```
link:example.com
```

## 7. 相关网站

使用 `related:` 查找相关网站：

```
related:github.com
```

## 8. 数字范围

使用 `..` 指定数字范围：

```
前端工资 10000..20000
```

## 9. 缓存版本

使用 `cache:` 查看缓存版本：

```
cache:example.com
```

## 常用组合

```
# 搜索技术问题
site:stackoverflow.com javascript undefined

# 搜索官方文档
site:developer.mozilla.org CSS flexbox

# 搜索开源项目
site:github.com React component library

# 搜索教程
前端教程 filetype:pdf
```

## 高级搜索

直接访问 https://www.google.com/advanced_search 可以使用更强大的搜索界面。
