# VSCode 中 Git 的 GUI 操作

## 1. 打开 Git 面板

- 快捷键：`Ctrl + Shift + G`
- 或点击左侧源代码管理图标

## 2. 暂存更改

- 点击文件旁边的 `+` 号
- 或右键选择 "暂存更改"

## 3. 提交

1. 在消息框中输入提交信息
2. 点击 `√` 或按 `Ctrl + Enter`

## 4. 切换分支

1. 点击底部状态栏的分支图标
2. 选择 "创建新分支" 或 "切换到..."

## 5. 拉取和推送

- 点击 "同步更改" 按钮
- 或右键选择 "拉取" / "推送"

## 6. 查看差异

- 点击文件查看与上一次提交的差异
- 修改部分用红色/绿色高亮

## 7. 解决冲突

1. 打开有冲突的文件
2. VSCode 会用高亮显示冲突部分
3. 选择 "接受当前更改"、"接受传入更改" 或手动编辑

## 8. 查看提交历史

- 点击 "提交" 视图中的 "更多操作"
- 选择 "查看提交历史"

## 9. 隐藏更改

- 右键选择 "暂存"
- 用于临时保存当前工作进度

## 10. GitLens 扩展推荐

安装 GitLens 扩展可获得更强大的功能：
- 行级历史追溯
- 提交者信息显示
- 丰富的比较视图
