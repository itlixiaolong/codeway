## 阶段一：2013-2015 入门期

### 核心技术栈

| 技术 | 掌握程度 | 学习建议 |
|-----|---------|---------|
| HTML/CSS | ⭐⭐⭐⭐⭐ | 重点学习布局（Flexbox/Grid） |
| JavaScript | ⭐⭐⭐ | 理解原型链、闭包、异步 |
| jQuery | ⭐⭐⭐⭐ | 学习 DOM 操作与插件开发 |
| Photoshop | ⭐⭐ | 前端需要的基本技能 |

### 我的入门经验

```javascript
// 回顾当年写过的代码
function initSlider() {
  // jQuery 时代的轮播组件
  var $slides = $('.slide');
  var currentIndex = 0;

  setInterval(function() {
    $slides.eq(currentIndex).fadeOut();
    currentIndex = (currentIndex + 1) % $slides.length;
    $slides.eq(currentIndex).fadeIn();
  }, 3000);
}
```

::: tip 给新人的建议
1. **不要急于学框架**：先把 JavaScript 基础打扎实
2. **多写原生代码**：理解浏览器原生 API 的工作原理
3. **动手实践**：每个知识点都要实际敲一遍代码
4. **阅读源码**：从 jQuery、Zepto 等库开始
:::

### 给年轻工程师的建议

::: warning 重要提醒
1. **不要追逐热门**：深入理解比广撒网更重要
2. **重视基础**：数据结构、算法、网络原理
3. **项目驱动**：在实战中学习和成长
4. **持续输出**：写博客、做分享、贡献开源
5. **关注业务**：技术最终要为业务创造价值
:::
